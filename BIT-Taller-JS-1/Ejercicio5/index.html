Instrucciones de uso

Para subir pasajero es necesario ingresar el nombre, cedula y seleccionar el vagon a viajar

Para bajar un pasajero solo debe colocar la cedula

Para buscar un pasajero puede usar Ctrl+F, pero como somos programadores, solo ingrese la cedula y presione el boton buscar
